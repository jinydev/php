|내부함수| 현재시간
```
int time ( void )
```

현재의 시간은 Unix epoch 이후의 시간을 timestamp 형식의 숫자 값으로 표기가 되는데 이를 가독성 있도록 표기를 하기 위해서는 date() 포맷함수를 이용하여 출력합니다. 

```php
<?php
	echo "현재의 시간은 = " . time() . "입니다. <br>";
	// 두번째 인자로 주어진 시간을 포맷에 맞게 출력합니다.
	echo date("H:i:s",time());
?>
```

화면출력)
```
현재의 시간은 = 1497766932입니다.
06:22:12
```
