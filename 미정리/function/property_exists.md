# property_exists()

클래스 내에서 지정한 프로퍼티가 존재 여부를 확인합니다.



```
property_exists($this, 'redirectPath');
```

위의 예제는 자신의 인스턴스인 `$this`에서 `redirectPath` 이름의 프로퍼티가 존재하는 지를 확인합니다.



외와 관련된 보다 자세한 정보는 

 http://php.net/manual/en/function.property-exists.php

에서 확인이 가능합니다.
