PHP 언어

php 언어는 1997년 처음으로 세상에 발표되어 현재까지 웹시장에서 가장 사랑을 많이 받는 언어중의 하나입니다.
오랜 기간동인 웹시장을 주도하고, 수많은 기업과 웹사이트들이 php 언어를 통하여 제작이 되고 있습니다.

서버사이드 웹프로그래밍

PHP는 동적 웹사이트를 제작할 수 있는 서버사이트 웹 프로그래밍 언어 입니다. 페이스북, 마젠토, cafe24등 수많은 서비스들이 PHP 언어로 운영되고 있습니다.

모던 PHP

PHP는 오랜 시간동안 수많은 유지 되어 왔습니다. 하위 호환성을 유지하기 위해서 몇년간 발전이 더딘적도 있지만, 현재는 모던 웹사이트 개발을 주도하기 위해서 빠른 개선작업들이 이루어 지고 있습니다. 모던스타일의 객체지향 프로그래밍, OP코드/JIT등의 도입으로 빠른 서비스의 성능 향상이 이루어 지고 있습니다.
