기호

PHP와 더불어 많은 프로그램 언어들은 문법을 구성하고 구분을 하는 기호를 사용합니다.
PHP 인터프리터는 C 언어로 개발되었고, 또한 문법적 표현과 기호들도 C언어와 유사한 기호들을 사용합니다.
PHP 언어를 작성하기 위해서 사용하는 몇 가지 기호들에 대해서 먼저 살펴보도록 하겠습니다.
