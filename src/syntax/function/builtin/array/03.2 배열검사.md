
03.2 배열검사
====================

배열은 여러 개의 데이터를 포함하고 있습니다. 같은 유형의 데이터의 묽음을 처리하는데 있어서 값을 검사하고 처리하데 매우 유용합니다.

PHP는 배열의 데이터를 검색하고 처리하는 몇 개의 내부함수들을 지원합니다.

03.2.1 검사
====================

내부함수 in_array()는 배열에 값이 존재하는지 검사를 할 수 있습니다. 값이 있는 경우에는 true 값을 반환합니다.

|내부함수| 배열 검색
bool in_array ( mixed $needle , array $haystack [, bool $strict = FALSE ] )

예제파일) array-23.php
<?php
	$arr = array("blue"=>"파랑색","red"=>"발강색","black"=>"검정색");

	if(in_array("파랑색",$arr)){
		echo "값이 존재합니다.";
	} else {
		echo "배열 값이 없습니다.";
	}
?>

화면출력)
값이 존재합니다.


|내부함수| 배열 인텍스키 확인
bool array_key_exists ( mixed $key , array $array )

내부함수 array_key_exists()는 연상배열안에 인텍스 키값이 존재하는지를 확인합니다.

예제파일) array-24.php
<?php
	$array = array('first' => null, 'second' => 4);

	// returns true
	if (array_key_exists('first', $array)){
		echo "배열에 키 값이 존재합니다.";
	} else {
		echo "키값이 없습니다.";
	}
?>

화면출력)
배열에 키 값이 존재합니다.

|내부함수|
mixed array_search ( mixed $needle , array $haystack [, bool $strict = false ] )

내부함수 array_search()는 지정한 배열에서 값을 검색하여 일치하는 키값을 반환합니다.

예제파일) array-25.php
<?php
	$array = array(0 => 'blue', 1 => 'red', 2 => 'green', 3 => 'red');

	$key = array_search('green', $array);
	// $key = 2;
	print_r($key);
	echo "<br>";

	$key = array_search('red', $array);
	// $key = 1;
	print_r($key);
?>

화면출력)
2
1


03.2.2 중복
====================

내부함수 array_count_values()는 배열안에 들어 있는 동일한 값의 개수를 반환합니다.

|내부함수| 중복값 체크
array array_count_values ( array $array )

예제파일) array-26.php
<?php
	$arr = array('orage', 'apple', 'grape', 'orage', 'apple', 'grape', 'orage');

	$arr_count = array_count_values($arr);
	while(list($key,$val) = each($arr_count)){
		echo $key. "==>" .$val. "<br>";
	}

?>

화면출력)
orage==>3
apple==>2
grape==>2


|내부함수| 중복값 제거
array array_unique ( array $array [, int $sort_flags = SORT_STRING ] )

내부함수 array_unique()는 입력된 배열값 중에서 중복된 내용은 제거합니다.

예제파일) array-27.php
<?php
	$arr = array("a" => "green", "red", "b" => "green", "blue", "red");
	$result = array_unique($arr);
	print_r($result);
?>

화면출력)
Array ( [a] => green [0] => red [1] => blue ) 

<br>

03.2.3 콜백 기반 최신 검색 및 검사 함수 (PHP 8.4+ / 8.5+)
====================

기존 PHP의 배열 검사는 단순 값 매칭(`in_array`)이나 키 확인(`array_key_exists`)에 의존했기 때문에, 복잡한 조건(예: 특정 값보다 큰 요소 찾기 등)으로 검색하려면 `foreach` 루프를 직접 작성해야 했습니다.
PHP 8.4 및 8.5에서는 JavaScript의 `find`, `some`, `every` 등과 유사하게 콜백 함수를 지정해 배열 요소를 간편하게 검사하고 찾아내는 빌트인 함수들이 대거 도입되었습니다.

### 1) array_find() — [PHP 8.4+]
배열 내에서 콜백 함수가 `true`를 반환하는 첫 번째 **요소의 값**을 구합니다. 매칭되는 항목이 없으면 `null`을 반환합니다.

|내부함수| 조건 만족 첫 번째 값 반환
mixed array_find(array $array, callable $callback)

예제파일) array-find.php
{% raw %}
```php
<?php
    $numbers = [5, 12, 8, 130, 44];
    
    // 10보다 큰 첫 번째 숫자를 찾습니다.
    $found = array_find($numbers, function($val) {
        return $val > 10;
    });

    echo "10보다 큰 첫 번째 값: " . $found;
?>
```
{% endraw %}

화면출력)
10보다 큰 첫 번째 값: 12


### 2) array_find_key() — [PHP 8.4+]
배열 내에서 콜백 함수 조건에 만족하는 첫 번째 **요소의 키(Key)**를 반환합니다. 만족하는 요소가 없으면 `null`을 반환합니다.

|내부함수| 조건 만족 첫 번째 키 반환
mixed array_find_key(array $array, callable $callback)

예제파일) array-find-key.php
{% raw %}
```php
<?php
    $users = [
        'user_1' => ['age' => 18],
        'user_2' => ['age' => 25],
        'user_3' => ['age' => 30]
    ];

    // 나이가 20세 이상인 첫 번째 유저의 키를 찾습니다.
    $key = array_find_key($users, function($user) {
        return $user['age'] >= 20;
    });

    echo "20세 이상 첫 번째 유저 키: " . $key;
?>
```
{% endraw %}

화면출력)
20세 이상 첫 번째 유저 키: user_2


### 3) array_any() 및 array_all() — [PHP 8.4+]
- **`array_any()`**: 배열 요소를 순회하며 **단 하나라도** 콜백 조건을 만족하면 `true`를 반환합니다. (JS의 `some`에 해당)
- **`array_all()`**: 배열의 **모든** 요소가 콜백 조건을 완벽히 만족해야만 `true`를 반환합니다. (JS의 `every`에 해당)

|내부함수| 하나라도 조건을 만족하는지 여부
bool array_any(array $array, callable $callback)

|내부함수| 모든 요소가 조건을 만족하는지 여부
bool array_all(array $array, callable $callback)

예제파일) array-any-all.php
{% raw %}
```php
<?php
    $scores = [85, 90, 78, 92, 88];

    // 90점이 넘는 점수가 하나라도 있는지 확인
    $hasExcellent = array_any($scores, function($score) {
        return $score >= 90;
    });

    // 모든 점수가 60점 이상(합격)인지 확인
    $allPassed = array_all($scores, function($score) {
        return $score >= 60;
    });

    echo "90점 이상 존재 여부: " . ($hasExcellent ? '예' : '아니오') . "<br>";
    echo "모두 합격 여부: " . ($allPassed ? '예' : '아니오');
?>
```
{% endraw %}

화면출력)
90점 이상 존재 여부: 예
모두 합격 여부: 예


### 4) array_first() 및 array_last() — [PHP 8.5+]
배열의 첫 번째 또는 마지막 요소를 반환합니다. 만약 배열이 비어있다면 `null`을 반환합니다.
기존에 `reset()`이나 `end()` 함수를 사용하여 내부 포인터를 조작하여 첫/끝 값을 가져오던 번거롭고 부작용이 많은 방식에서 벗어나, 깔끔하고 직관적으로 사용할 수 있는 표준 내장 함수입니다.

|내부함수| 배열의 첫 번째 값 반환
mixed array_first(array $array)

|내부함수| 배열의 마지막 값 반환
mixed array_last(array $array)

예제파일) array-first-last.php
{% raw %}
```php
<?php
    $fruits = ["사과", "바나나", "포도", "오렌지"];

    $first = array_first($fruits);
    $last = array_last($fruits);

    echo "첫 번째 과일: " . $first . "<br>";
    echo "마지막 과일: " . $last;
?>
```
{% endraw %}

화면출력)
첫 번째 과일: 사과
마지막 과일: 오렌지
