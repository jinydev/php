# 모던 PHP
---
PHP는 오랜 시간동안 수많은 유지 되어 왔습니다. 하위 호환성을 유지하기 위해서 몇년간 발전이 더딘적도 있지만, 
현재는 모던 웹사이트 개발을 주도하기 위해서 빠른 개선작업들이 이루어 지고 있습니다. 모던스타일의 객체지향 프로그래밍, 
OP코드/JIT등의 도입으로 빠른 서비스의 성능 향상이 이루어 지고 있습니다.
